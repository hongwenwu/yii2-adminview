# yii2-adminview
Yii2 开源后台模板。使用非常简单，只需要简单几步就可完成！

安装
----

安装这个扩展的首选方法是通过 [composer](http://getcomposer.org/download/)。

可以运行

```
composer require --prefer-dist yiioctopus/yii2-adminview "*"
```

也可以添加

```
"yiioctopus/yii2-adminview": "*"
```

到你的 `composer.json` 文件的包含部分。
